# React, Redux, Router, Axios, Immutable, ESLint, ES6, Sass Boilerplate

## Install
```sh
$ npm install -g eslint eslint-plugin-jsx-a11y@^2.2.3 eslint-plugin-import eslint-plugin-react eslint-config-airbnb
$ npm install
```

## Run
```sh
$ npm start
```

## Build
```sh
$ npm run build
```
